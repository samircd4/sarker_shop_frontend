# Sarker Shop

> Modern e-commerce website built with React and Tailwind CSS.

## Features

- 🛍️ Product catalog
- 📱 Responsive design
- 🛒 Shopping cart
- 🗂️ Categories
- 🔍 Search functionality

## Tech Stack

- React 18
- React Router DOM
- Tailwind CSS
- DaisyUI
- Vite

## Installation

```bash
# Clone repository
git clone https://github.com/samircd4/sarker_shop_frontend.git sarker-shop

# Navigate to project
cd sarker-shop

# Install dependencies
npm install

# Start development server
npm run dev
```
